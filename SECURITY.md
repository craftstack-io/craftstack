# Security Policy

## Melaporkan Kerentanan (Vulnerability)

Karena CraftStack adalah proyek statis (HTML/CSS/JS tanpa backend), risiko keamanan yang umum terkait dengan:

- Kerentanan XSS pada input form (login, register, newsletter)
- Dependency pihak ketiga (Google Fonts CDN)

Jika kamu menemukan celah keamanan, mohon **jangan** membuka issue publik. Sebagai gantinya:

1. Kirim laporan melalui GitHub Security Advisory di tab **Security** repo ini, atau
2. Hubungi maintainer secara langsung melalui email/kontak yang tertera di profil GitHub

Sertakan:
- Deskripsi kerentanan
- Langkah reproduksi
- Dampak potensial

## Response Time

Kami akan berusaha merespons laporan dalam **7 hari kerja**.
