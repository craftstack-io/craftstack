# Contributing to CraftStack

Terima kasih sudah tertarik untuk berkontribusi! 🎉

## Cara Berkontribusi

1. **Fork** repository ini ke akun GitHub kamu
2. **Clone** hasil fork ke lokal:
   ```bash
   git clone https://github.com/<username-kamu>/CraftStack.git
   cd CraftStack
   ```
3. **Buat branch baru** untuk perubahanmu:
   ```bash
   git checkout -b feat/nama-fitur
   ```
4. Lakukan perubahan, lalu **test** dengan membuka `index.html` di browser
5. **Commit** dengan pesan yang jelas:
   ```bash
   git commit -m "feat: tambahkan animasi baru pada hero section"
   ```
5. **Push** ke fork kamu:
   ```bash
   git push origin feat/nama-fitur
   ```
6. **Buka Pull Request** ke branch `main` repo ini

## Panduan Gaya Kode

- Gunakan indentasi 2 spasi
- Ikuti konvensi penamaan CSS custom properties yang sudah ada di `:root`
- JavaScript ditulis vanilla (tanpa framework/library eksternal)
- Pastikan perubahan tetap responsive (mobile & desktop)

## Melaporkan Bug

Gunakan template [Bug Report](.github/ISSUE_TEMPLATE/bug_report.md) saat membuka issue.

## Mengusulkan Fitur

Gunakan template [Feature Request](.github/ISSUE_TEMPLATE/feature_request.md).

## Kode Etik

Dengan berkontribusi, kamu setuju untuk mengikuti [Code of Conduct](./CODE_OF_CONDUCT.md) kami.
